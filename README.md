# Experiments on Deep Learning Models for ECG-Based Arrhythmia Detection
Data set used : MIT-BIH Arrhythmia Dataset.
he following models were implemented and evaluated:

CNN Baseline Model for ECG classification

CNN + BiLSTM to capture spatial and temporal features

1D ResNet for deeper feature extraction from ECG signals

CNN + BiLSTM + Attention to improve model focus on important patterns
